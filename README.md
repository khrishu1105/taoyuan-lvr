# 桃園實登查詢站

桃園市不動產實價登錄查詢工具，涵蓋**預售屋、中古/成屋、土地**三大品項。純靜態網頁，資料來自內政部政府開放資料，透明可驗證。

> 貝多房地產內部工具 · 由 khrishu 建置

## 特色

- **三品項合一**：預售屋（建案聚合）、中古屋（逐筆＋區×型態統計）、土地（逐筆＋區×分區統計）
- **解約透明**：預售屋將「解約筆數」分開標示，`有效筆數 = 總登錄筆數 − 解約`，避免高估去化
- **口徑公開**：單價＝內政部「單價元/平方公尺」換算萬/坪（×3.305785）
- **自動更新**：`update.py` + macOS launchd 排程，每旬自動抓最新一批公開資料

## 檔案結構

```
index.html              查詢介面（三分頁，lazy-load）
data/*.json             前端資料（由 etl.py 產生）
etl.py                  ETL：raw/*.zip → SQLite + JSON
update.py               自動更新：重抓最近季別 → 重建
build_standalone.py     打包單機版 HTML（lite / full）
開啟查詢頁.command       Mac 一鍵啟動本機伺服器
com.beido.*.plist       launchd 排程設定
```

## 本機執行

```bash
# 方式一：雙擊「開啟查詢頁.command」（Mac）
# 方式二：
python3 -m http.server 8777
# 瀏覽器開 http://localhost:8777/index.html
```

## 更新資料

```bash
python3 update.py     # 重抓最近季別、重建資料庫與 JSON
```
